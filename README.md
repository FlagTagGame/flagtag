# FlagTag
An Online Multiplayer Capture The Flag Game

## Tech Stack
- Phaser.js
- Node.js
- Matter.js
- Express.js
- JavaScript

## Running on your local machine
### Required Tools:
- Git
- Node.js
```
$ git clone https://github.com/Hydrosaur/flagtag.git
$ cd flagtag
$ npm i
$ npm test
```
Then go to `localhost:8080/game` on your browser.
